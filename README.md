
# Stock-app

A simple stock-app made with TypeScript, express.js and MongoDB

## Run Locally

Clone the project

```bash
  git clone https://github.com/BjornFrancke/stock-app
```

Go to the project directory

```bash
  cd stock-app
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
